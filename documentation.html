Parsing Context-Free Grammars

Introduction

Welcome to the Team Chill LL(1) Parser! This project implements an LL(1) parser for context-free grammars. LL(1) parsers are top-down parsers that use a single lookahead symbol to make parsing decisions. This parser analyzes a string of symbols and checks whether it can be derived from a given context-free grammar, displaying the result as Accepted or Rejected.

Features

1. Grammar Input
2. Start Symbol Input
3. Generate Parsing Table
4. Parse String
5. Reset Functionality

1. Grammar Input: You can define your grammar by entering one rule per line in the format: 

              Non-terminal -> Right-hand side symbols

2. Start Symbol Input: Enter the start symbol for the grammar, which is the non-terminal from which the parsing process begins.

3. Generate Parsing Table: Click the "Generate Parser" button to automatically generate the LL(1) parsing table based on the grammar input.

4. Parse String: Input a string to be parsed, and the parser will analyze whether the string can be derived from the grammar.

5. Reset Functionality: Use the reset button to clear all inputs and results for a fresh start.

Implemented Methodology

1. **First Set Computation**
2. **Follow Set Computation**
3. **Parse Table Construction**
4. **Top-Down Parsing Process**

Example

Here's an example of how to define a simple grammar and parse a string:

Grammar: 

            S -> A B
            A -> a
            A -> ε
            B -> b

Start Symbol: `S`

Input String: `a b`

Parsing Table

          | Non-Terminal | Terminal | Production |
          |--------------|----------|------------|
          | S            | a        | S -> A B   |
          | A            | a        | A -> a     |
          | B            | b        | B -> b     |

Result: Accepted

Conclusion

The Team Chill LL(1) Parser is an efficient tool for parsing context-free grammars using the LL(1) parsing method. It allows users to input a grammar, generate a parsing table, and parse strings to check if they are accepted by the grammar. Whether you're learning parsing algorithms or building parsers for specific applications, this tool provides a simple yet powerful approach to syntactic analysis.
